# 📊 Online Retail Dashboard - Power BI Project

## 🚀 Overview

This project presents an interactive **Online Retail Dashboard** built using Power BI to analyze sales performance, customer behavior, and product trends.

The dashboard transforms raw data into meaningful insights for better business decision-making.

---

## 📌 Key Features

* 📈 Revenue analysis by **Month & Day**
* 🌍 Country-wise **sales and customer distribution**
* 📦 Product-wise **revenue insights**
* 📊 KPI Metrics:

  * Total Revenue
  * Total Customers
  * Total Quantity
  * Total Invoices
* 🎯 Interactive filters (Country, Month, Year)

---

## 🛠 Tools & Technologies

* Microsoft Power BI
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation
* Data Visualization Techniques
